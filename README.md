# AngularElectron

Build a native desktop timer app with Angular and Electron 

![](https://github.com/YannMjl/Library-Space/blob/master/img/angular-electron.gif)

- Angular v4.2
- Electron v1.7
- Angular CLI v1.4 

## Basic Usage

```shell
git clone
cd angular-electron
npm install

# build the app
npm run electron-build
```
